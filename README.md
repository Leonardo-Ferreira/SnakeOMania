# Snake O'Mania

Snake O'Mania is classical snake game rewrite on C# using .Net Core 3.1+

## Desired Features
1. Good'old single player
2. Difficulty settings
3. Special Apples
4. Multiplayer with Dedicated server
5. A nice GUI
6. Online leaderboard
7. Single Player vs A.I.
8. Game Recording
